# Twitchbot Dashboard

A simple dashboard for [Twitchbot](https://github.com/Dilaz/Twitchbot-dashboard).

## Frontend

### Config
Add `VUE_APP_API_URL` to env, for example `http://localhost:3000`

### Running

```bash
npm install
npm run serve
```

## Backend

### Config
Copy `.env.example` to `.env` and add your own keys

### Running

```bash
npm install
npm run start:dev
```
